## Forms Section

The **Forms** section explains how to build user-friendly forms using Bootstrap’s styles and utilities.

### Commonly Used Form Classes

| Form Element            | Bootstrap CSS Class |
| ----------------------- | ------------------- |
| `input`                 | `form-control`      |
| `input type="checkbox"` | `form-check-input`  |
| `input type="radio"`    | `form-check-input`  |
| `input type="range"`    | `form-range`        |
| `select`                | `form-select`       |

These classes ensure consistent styling across different input types and states.

---

## Switches

Bootstrap allows checkbox inputs to be styled as switches.

### How to Use

* Wrap the input in a `div`
* Apply `form-check` and `form-switch` to the div
* Apply `form-check-input` to the input element

---

## Input Groups

**Input groups** allow extra content to be attached before or after an input field.

### Common Use Case

* Displaying currency symbols or units alongside inputs

### Structure

* Wrap content in a `div` with `input-group`
* Use `span` elements with `input-group-text`
* Place input inside the group

---

## Floating Labels

Floating labels provide contextual information inside form fields.

### Features

* Label stays visible when the input is focused or contains text
* Replaces traditional placeholders

### Usage

* Wrap input and label inside a `div` with `form-floating`
* Use `form-control` on the input
